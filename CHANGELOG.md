# 更新日志

本项目的所有重要变更均记录于此，格式参考 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)。

## [0.3.0] - 2026-05-31

### 新增
- 点餐页点击菜品弹出详情，展示原材料与烹饪方式
- 录入/编辑表单支持「🌐 从网址获取」烹饪方法（抓取网页后由 AI 提炼）
- 系统内「项目文档」页：需求文档、更新日志、可视化接口文档（含在线调试）

### 变更
- **移除菜品价格**：菜单、购物车、订单均不再展示或计算价格，改为统计件数
- AI 配置改为仅通过环境变量 `AI_API_KEY` / `AI_BASE_URL` / `AI_MODEL` 提供，不再硬编码密钥

### 修复
- 订单创建逻辑适配无价格数据模型

---

## [0.2.0] - 2026-05-31

### 新增
- **随机轮盘**：按上架菜品分扇区随机抽取，带转动音效、扇区高亮、抽中后自动加入购物车并跳转点餐页
- **汤品类型**：支持菜品 / 汤品分类，内置示例「紫菜蛋花汤」
- **拍照录入**：摄像头拍照或上传图片作为菜品图
- **烹饪方法**：手动录入或「✨ AI 推荐」生成
- **原材料手动关联**：录入时可新建原材料并即时关联
- 购物车跨页签共享（`store.js` + `menu.js`）

---

## [0.1.0] - 2026-05-31

### 新增
- 初版小馆点餐系统（Golang + Vue3）
- 点餐：浏览上架菜品、购物车、下单（按配方扣减原材料库存）
- 菜品管理：上下架、原材料配置
- 原材料 / 库存：有限库存与无限库存（`stock = -1`）
- 订单记录：状态流转（待处理 / 已支付 / 完成 / 取消）
- REST API + SQLite 持久化 + 演示种子数据
